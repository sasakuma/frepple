===============
ERP integration
===============

This chapter describe frePPLe's capabilities to integrate with
external ERP systems. As of today, connectors exist with Odoo, Openbravo,
ERPNext and uzERP.

Your ERP is not in the above list?
You can use the generic ERP connector, this is a squeleton piece of code that needs
to be tailored to match your ERP data model.
